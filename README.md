# ChatPGT 🧠

---

ChatPGT is a very simple open-source neural network written in C#.

The project has several goals:
- Learn how neural networks are structured and how they work.
- Write a neural network that can recognize handwritten digits from the MNIST dataset.

As the project progresses, the neural network will be able to perform various tasks.  
The project will be considered complete as soon as it learns to distinguish between digits.
