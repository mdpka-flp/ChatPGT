# ChatPGT 🧠

---

ChatPGT - это простейшая открытая нейросеть, написанная на C#.

У проекта есть несколько целей:
- изучить, как устроены и как работают нейросети;
- написать нейросеть, которая сможет различать рукописные цифры из датасета MNIST.

As the project progresses, the neural network will be able to perform various tasks  
The project will be completed as soon as it learns to distinguish between numbers

По мере прогресса проекта, нейросеть сможет выполнять разные задачи  
Проект будет окончен как только она научиться различать цифры
